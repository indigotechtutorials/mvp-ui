# MVP UI

A minimal css library installable with one file.

### Setup guide

To run the app and view the components you can download the source code

##### Step 1. Download source code

`git clone https://github.com/indigotechtutorials/mvp-ui`

`cd mvp-ui`

##### Step 2. Start web server

`ruby server.rb`

##### Step 3. View website in browser

go to [http://localhost:3000](http://localhost:3000)
